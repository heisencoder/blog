Outline:<br />
<ul>
<li> Monitor Size: 30 inch with 2560x1600 resolution</li>
<ul>
<li>Single monitor vs. Dual</li>
<ul>
<li>Neck strain from looking at multiple monitors</li>
<li>Mac is not setup well for multiple monitors (e.g., menu bar only on one screen) </li>
<li>Dual-monitors have an awkward break in the middle</li>
</ul>
<li>You can have too much screen real-estate:</li>
<ul>
<li>Programmers can only productively focus on one thing at a time</li>
<li>Use Workspaces (multiple desktops) and switch between isolated tasks</li>
<li>Site Jeff Atwood as a counter-example </li>
</ul>
<li>25 inch vs. 30</li>
<ul>
<li>Budget considerations: 25 inch good value</li>
<li>Eye sight considerations: Glasses should frame the monitor, get computer glasses: optimized for </li>
<li>Dual-link DVI vs. Single link</li>
</ul>
</ul>
<li>Standing Desk vs. Sitting</li>
<li>Monitor height</li>
<li>Mouse on the left side</li>
<li>Using multiple computers:</li>
<ul>
<li>Route all input to one monitor and keyboard</li>
<li>Use a KVM switch (that supports dual-link)</li>
<li>Use virtual desktops:</li>
<ul>
<li>rdesktop, NX Viewer, VNC</li>
</ul>
</ul>
</ul>
