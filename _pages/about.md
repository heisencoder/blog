## About This Blog

Heisencoder is a technical blog covering software development, cryptography, security, and programming languages. The blog was originally started in 2007 and ran through 2012 on Blogger.

### Topics Covered

This blog explores:

- **Cryptography & Security**: Key management, encryption standards (IEEE P1619), full-disk encryption, security protocols
- **Programming Languages**: Ada, Ruby, Python, and discussions about language design
- **Developer Tools**: IDE customizations (particularly for Mac users), keyboard shortcuts, syntax highlighting
- **Software Engineering**: Algorithms, data structures, best practices, and software architecture

### About the Author

Matthew V Ball is a software engineer with interests in security, cryptography, and developer productivity tools. Many of the early posts came from working on security standards and encryption technologies.

### Blog History

- **2007-2012**: Originally published on Blogger at heisencoder.net
- **2025**: Migrated to GitHub Pages with modern Jekyll theme
- All original posts and comments have been preserved
